# Current Spec

## 概要

このゲームは、ブラウザで直接動かせる 2D アクションシューティングです。  
現状は `Standard`、`Duel`、`Siege` の3モードがあり、プレイヤーの高機動アクションとマウス照準射撃を中心に構成されています。

## 基本ルール

- スコアアタック形式
- 時間経過で敵が強化される
- 敵弾、接触ダメージあり
- 被弾や立ち回りの難しさと、高機動の得点効率を両立する設計

## 操作

- `A / D`
  - 左右移動
- `W / Space`
  - ジャンプ
- `Space` 長押し
  - ブースト上昇
- `Shift`
  - しゃがみ
- `Shift + 左右`
  - ローリング
- 着地硬直中 `Shift + 左右`
  - ローリングキャンセル
- 着地硬直中 `Shift + W`
  - ハイジャンプ
- 空中 `Shift + W`
  - 二段ジャンプ
- 空中 `Shift + 左右`
  - エアダッシュ
- 空中 `Shift + S`
  - 急降下
- `左クリック`
  - 射撃
- `右クリック`
  - 近接攻撃
- `R`
  - リスタート
- `Esc`
  - タイトルへ戻る

## プレイヤーアクション

### ジャンプ

- `W / Space` で通常ジャンプ
- `Space` はジャンプ後そのままブースト入力にも使える

### ブースト

- 空中で上向き推力を発生
- ブーストゲージを消費
- ゲージ切れ後は一定時間回復停止
- 落下緩和や空中制動にも使用可能

### ローリング

- `Shift + 左右` で発動
- 慣性で滑る
- 逆方向入力で減速
- ローリング中は射撃不可
- ローリング中は完全無敵
- ローリングジャンプで横っ飛び可能

### 着地硬直

- 速い落下後は着地硬直あり
- 通常着地硬直はローリングキャンセル可能
- 急降下キック着地硬直はローリングキャンセル不可

### 二段ジャンプ

- 空中で `Shift + W`
- 上昇中に使うと少し伸びる
- 上向き慣性がない時でも十分に跳ねるよう最低上昇速度あり

### エアダッシュ

- 空中で `Shift + 左右`
- 横方向の高速移動
- 受付中に飛び蹴り派生可能

### 急降下

- 空中で `Shift + S`
- 横慣性を切りつつ下方向へ高速移動
- 受付中に急降下キック派生可能

### ハイジャンプ

- 着地硬直中に `Shift + W`
- 強く上方向へ飛ぶ

## 近接アクション

### 通常キック

- `右クリック`
- 攻撃方向は移動方向ではなく、照準の左右方向
- 射程、威力、吹き飛ばしは `CONFIG.melee` で管理

### 飛び蹴り

- エアダッシュ中に `右クリック`
- 命中で敵を大きく吹き飛ばす
- 命中後、急降下再使用可能

### 急降下キック

- 急降下中に `右クリック`
- 敵命中で跳ね返る
- 地面着地で衝撃波発生
- 命中後、エアダッシュ再使用可能
- 衝撃波着地時は短めの硬直あり

## 能力解放順

空中アクションは以下の順に解放されます。

1. `airDash`
2. `doubleJump`
3. `dive`
4. `highJump`
5. `dashKick`
6. `diveKick`

## 武器システム

### 基本

- 持ち替え式ではなく進化式
- `W CORE` を集めると武器段階が進む
- 10個ごとに1段階進化

### 武器進化

- 初期武器は弱め
- 段階が進むごとに火力・弾の性質が変化
- 最終武器は3wayだが、扇状ではなく照準へ収束する3列射撃

## アイテム

### Weapon Cache

- 敵ドロップ
- `W CORE` を1加算

### Mobility Chip

- 敵ドロップ
- 次の空中能力を1段階解放

### Repair Pack

- 敵ドロップではない
- 足場に時間経過で出現
- プレイヤーHPが減っている時に取得で回復

## 回復アイテム出現仕様

- 出現場所
  - 足場の上
- 出現方式
  - 全体でおおよそ10秒に1個
- 同時出現上限
  - 開始から3分未満: 3個
  - 3分以降: 2個
- 狙い
  - 地上待ち一択を避け、足場移動の意味を持たせる

## 敵システム

### 共通

- 時間経過でHP増加
- 弾を撃つ
- 接触ダメージあり
- ノックバック中は接触ダメージ無効
- 出現前に予告表示あり

### 敵種

- `charger`
  - 接近型
- `standoff`
  - 距離を取って撃つ
- `drone`
  - 空中移動型
  - ヘッドショット無効
- `rival`
  - `Duel` 専用の高機動敵

## ヘッドショット

- 人型敵のみ有効
- 頭と胴体で判定分離
- ドローンはヘッドショット無効
- 接触ダメージや近接でヘッドショット扱いにはならない

## 移動コンボ

- 対象アクション
  - `roll`
  - `sideLeap`
  - `airDash`
  - `dive`
  - `landingCancel`
  - `flyingKick`
  - `doubleJump`
  - `highJump`
- 一定時間内につなぐと `MOVE COMBO` 増加
- コンボ中に射撃を当てるとスコアボーナス
- 被弾でコンボリセット

## モード

### Standard

- 基本のサバイバルモード

### Duel

- 高機動ライバルとの1対1寄りモード
- 通常より敵射撃頻度高め
- ライバル専用スポーン位置あり
- 足場構成も専用化

#### Duel の足場

- 一番下の右足場なし
- 一番下の左足場なし
- 通常より下段が薄い構成

#### Duel のライバル

- ジャンプ
- 空中移動
- ダッシュ
- ローリング

#### Rival Roll

- 一定条件で使用
- ロール中は無敵
- ロール中は接触ダメージ無効
- ロール中は射撃しない

### Siege

- 射撃圧と火力を高めたモード

## ステージ

### 通常

- 横長ステージ
- 床 + 複数足場

### Duel

- 通常より足場を削った専用レイアウト
